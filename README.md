# A CNode.org ios client App.


## development env

    * mac os 10.12
    * xcode v8.3
    * cocoaPod
    * swift package manager (SPM)

## development flow

    * git clone git@github.com:FanHehe/CNode-ios.git
    * cd CNode-ios
    * pod install
    * pod update
    * open fanhehe.xcworkspace
    * command + b && command + r
